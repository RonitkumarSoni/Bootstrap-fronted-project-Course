# WsCube Tech Clone - Bootstrap 5 Project

## Overview
This is a fully responsive frontend clone of the WsCube Tech website. It is built using HTML5, CSS3, and Bootstrap 5 to demonstrate modern web development practices and responsive design.

## Features
- Fully responsive layout (Mobile, Tablet, Desktop)
- Interactive Offcanvas mobile menu
- Enquiry and Demo booking Modals
- FAQs Accordion
- Course Highlights & Tabbed Curriculum

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Setup Instructions
1. Clone the repository: `git clone https://github.com/RonitkumarSoni/Bootstrap-fronted-project-Course.git`
2. Open `index.html` in any modern web browser.
3. No additional setup or package installation is required.
