# WsCube Tech Clone - Bootstrap 5 Project

## Project Overview
This project is a fully responsive frontend clone of the WsCube Tech website. It is built using HTML5, CSS3, and Bootstrap 5. The purpose of this project is to demonstrate proficiency in modern web development practices, responsive design, and the effective use of the Bootstrap framework. It solves the problem of creating a visually appealing and functional educational platform landing page.

## Features
- Fully responsive layout for mobile, tablet, and desktop.
- Interactive Offcanvas mobile menu.
- Enquiry and Demo booking Modals.
- Testimonial counters.
- Course highlights with icons.
- Interactive Accordion for FAQs.
- Tabbed navigation for Curriculum sections.
- Comprehensive footer with multi-column links.

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla for basic logic)
- FontAwesome (Icons)

## Bootstrap Concepts Used
- **Container**: Used `.container` and `.container-fluid` for wrapping content and centering.
- **Grid System**: Used `.row` and `.col-*` classes to create responsive grid layouts.
- **Rows & Columns**: Building blocks of the grid system for horizontal groups of columns.
- **Breakpoints**: Used sizing classes (`-sm`, `-md`, `-lg`) to adjust layouts on different screens.
- **Responsive Design**: Ensuring the site looks good on all devices using Bootstrap's responsive utilities.
- **Navbar**: Used custom header but integrated Offcanvas for mobile menu.
- **Buttons**: Used `.btn`, `.btn-primary`, `.btn-outline-primary` for interactive buttons.
- **Forms**: Used `.form-label`, `.form-control`, `.input-group` for input fields.
- **Accordion**: Used `.accordion` for the FAQ section to toggle visibility of content.
- **Collapse**: Used for toggleable content like the FAQ answers.
- **Modal**: Used `.modal`, `.modal-dialog` for popup forms (Enquire, Book Demo).
- **Offcanvas**: Used `.offcanvas` for the slide-out mobile menu.
- **Navs & Tabs**: Used `.nav-pills` for the curriculum tabs.
- **Tooltip**: Initialized tooltips for the WhatsApp button.
- **Utilities**: Extensively used spacing, display, and flexbox utilities.
- **Flexbox Utilities**: `.d-flex`, `.justify-content-center`, `.align-items-center` for alignment.
- **Spacing Utilities**: `.m-*`, `.p-*`, `.my-5`, `.px-4` for margins and padding.
- **Display Utilities**: `.d-none`, `.d-lg-block` for showing/hiding elements responsively.
- **Position Utilities**: `.sticky-top`, `.position-relative` for element positioning.
- **Colors**: `.text-white`, `.bg-white`, `.bg-secondary-subtle` for text and background colors.
- **Shadows**: `.shadow-lg` for adding depth to elements.

## JavaScript Logic Used
- Initialization of Bootstrap components like Modals and Offcanvas via data attributes.
- Tooltip initialization using Bootstrap's JavaScript API.

## Common Interview Mistakes
- Not knowing the difference between Bootstrap 4 and 5 (e.g., removal of jQuery).
- Overriding Bootstrap classes with custom CSS instead of using variables or utility classes.
- Misunderstanding the grid system breakpoints.

## Best Practices
- Always use a container wrapper for row and col.
- Utilize utility classes before writing custom CSS.
- Mobile-first approach for custom styling.

## Cheat Sheet
- Grid: `.col-`
